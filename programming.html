HTML چیست؟

HTML (HyperText Markup Language) زبان پایه برای ساختار وب‌سایت است. این زبان عناصر مختلفی مثل عنوان، پاراگراف، لینک و تصاویر را مشخص می‌کند.

CSS چیست؟

CSS (Cascading Style Sheets) برای استایل‌دهی و ظاهر کردن صفحات وب استفاده می‌شود. با CSS می‌توان رنگ، فونت، اندازه و موقعیت المان‌ها را کنترل کرد.

body {
  background-color: #f5f9ff;
  color: #333;
}
h3 {
  color: #1a73e8;
}

JavaScript چیست؟

JavaScript زبان برنامه‌نویسی است که به صفحات وب هوش و تعامل می‌دهد. با JS می‌توان اقداماتی مانند تغییر محتوا، اعتبارسنجی فرم و واکنش به کلیک کاربر انجام داد.

let message = "سلام دنیا!";
console.log(message);
alert(message);
